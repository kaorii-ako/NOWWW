# WarioWare Game Implementation Plan

## Project: NOWWW (Godot 4.7)

**Goal**: Create a WarioWare-style microgame collection with 4 minigames, original winner/death scenes, and programmatic pixel art assets.

---

## Current State
- Godot 4.7 project exists with scaffolding (`global.gd`, `start_menu.gd`)
- No scenes (`.tscn`) exist yet
- Git initialized, no commits
- Remote: `https://github.com/kaorii-ako/NOWWW.git`

---

## Architecture

### Core Systems

1. **MinigameBase** (`scripts/minigame_base.gd`) - Abstract base class
   - `@export time_limit: float` + `instruction_text: String`
   - Signals: `won`, `lost`
   - Methods: `win()`, `lose()`, `_on_timeout()`
   - Built-in countdown timer + instruction display

2. **Game Manager** (`scripts/level.gd` + `scenes/level.tscn`)
   - Loads minigames one at a time from shuffled pool
   - Shows "GET READY" between minigames
   - Handles transitions to winner/game_over scenes

3. **Enhanced Global** (`global.gd`)
   - Add `minigame_pool: Array[String]` with scene paths
   - Add `get_next_minigame_path()` with shuffle logic
   - Win after 5 minigames, lose after 3 deaths

### Minigames (4 total)

| # | Name | Controls | Mechanic | Time |
|---|------|----------|----------|------|
| 1 | DODGE! | Arrow keys | Avoid falling blocks | 5s |
| 2 | TAP! | Mouse click | Click 5 targets, avoid bombs | 5s |
| 3 | STOP! | Spacebar | Stop indicator in green zone | 4s |
| 4 | MATCH! | Mouse click | Recreate shown pattern (3x3 grid) | 6s |

### Scenes

```
scenes/
├── start_menu.tscn      # Title + Start/Quit buttons
├── level.tscn           # Game hub, loads minigames
├── winner.tscn          # Victory celebration
├── game_over.tscn       # Death screen
└── minigames/
    ├── dodge_the_blocks.tscn
    ├── click_the_targets.tscn
    ├── stop_on_green.tscn
    └── match_the_pattern.tscn
```

### Assets Strategy
- **Programmatic generation** using `SpriteFactory` class
- Pixel art style with consistent color palette
- Simple beep sounds via `AudioFactory` or minimal `.wav` files
- Satisfies "implement own assets" requirement

## Winner Scene Design
- Bright yellow/white background
- Bouncing geometric shapes (squares, triangles)
- Title bounces in with elastic easing
- Score counter animates from 0 to final score
- "YAY!" sound on entry

## Game Over Scene Design
- Dark red background fading in
- Screen shake on entry
- Skull made of geometric rectangles
- Title appears with typewriter effect
- Brief white flash
