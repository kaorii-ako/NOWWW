# NOWWW

A WarioWare-inspired microgame collection built in Godot 4.7.

## How to Play

- Survive 5 minigames to win!
- You have 3 lives — lose them all and it's game over
- Each minigame lasts 4-6 seconds
- Read the instruction and react fast!

## Minigames

| Name | Controls | Objective |
|------|----------|-----------|
| DODGE! | Arrow keys | Avoid falling blocks for 5 seconds |
| TAP! | Mouse click | Click 5 targets, avoid bombs |
| STOP! | Spacebar | Stop the indicator in the green zone |
| MATCH! | Mouse click | Recreate the shown pattern |

## Built With

- Godot 4.7
- GDScript
- All assets created programmatically (pixel art style)

## Development

- Total development time: 5+ hours
- AI usage: Minimal (planning assistance only)

## Running

1. Clone the repository
2. Open in Godot 4.7
3. Press F5 to run

## Controls

- **Arrow Keys**: Move left/right in DODGE minigame
- **Mouse Click**: Select targets in TAP and MATCH minigames
- **Spacebar**: Stop the indicator in STOP minigame
- **Enter/Space**: Navigate menus
