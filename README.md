# NOWWW

A WarioWare-inspired microgame collection built in Godot 4.7.

## How to Play

1. **Start the Game**: Press the START button on the main menu
2. **Survive**: You must complete 5 minigames to win
3. **Lives**: You have 3 lives - lose them all and it's game over
4. **React Fast**: Each minigame lasts 4-6 seconds
5. **Read Instructions**: Each minigame shows what you need to do
6. **Win or Lose**: Complete the objective to advance, fail and lose a life

## Minigames

| Name | Controls | Objective | Time |
|------|----------|-----------|------|
| DODGE! | Arrow Keys | Avoid falling blocks | 5s |
| TAP! | Mouse Click | Click 5 targets, avoid bombs | 5s |
| STOP! | Spacebar | Stop indicator in green zone | 4s |
| MATCH! | Mouse Click | Recreate the shown pattern | 6s |

## Controls

- **Arrow Keys**: Move left/right in DODGE minigame
- **Mouse Click**: Select targets in TAP and MATCH minigames
- **Spacebar**: Stop the indicator in STOP minigame
- **Enter/Space**: Navigate menus
